Info

This application is intended to help developers experiment with the APIs at a high level without needing to know how to make API requests. It uses a proxy to allow requests to be made via JavaScript to the Bungie.net Platform.

Depending on which proxy you decide to use, you may be required to supply your API Key in order to make requests.

Authenticated requests are also supported however you are required to go obtain a recent "bungleatk" cookie from Bungie.net and is totally optional if you just wish to use the public APIs.

This application was developed in good faith and neither your API Key or Bungie.net session are tracked in any way. Use at your own risk.
