DestinyService / EquipItems

Edit Wiki

This page was generated with Wiki Builder. Do not change the format!

Info

Equips multiple items from a character's inventory and returns the updated inventory and character information. The endpoint will fail if the character is logged in and doing an activity.

  • URI: /Destiny/EquipItems/
  • Method: POST
  • Accessibility: Private
  • Service: DestinyService

Parameters

Path Parameters

None

Query String Parameters

None

JSON POST Parameters

Name Description
membershipType A valid Bungie.net membershipType.
characterId A valid characterId that is associated with the given account.
itemIds An array of item instanceIds to equip.

POST https://www.bungie.net/Platform/Destiny/EquipItems/
