ActivityService / GetFollowersOfUser

Edit Wiki

Info

Returns a list of Bungie.net users following a given user.

  • URI: /Activity/User/{i}/Followers/
  • Method: GET
  • Accessibility: Public
  • Service: ActivityService

Parameters

Path Parameters

Name Description
i  

Query String Parameters

Name Description
itemsperpage The number of items per page. Default is 50.
currentpage The current page. Starts a 1.

JSON POST Parameters

None

https://www.bungie.net/Platform/Activity/User/5197148/Followers/
