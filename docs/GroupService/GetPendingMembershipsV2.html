GroupService / GetPendingMembershipsV2

Info

Get a paged list of pending members for a given group.

  • URI: /Group/{groupId}/Members/PendingV2/
  • Method: GET
  • Accessibility: Public
  • Service: GroupService

Parameters

Path Parameters

Name Description
groupId A valid groupId.

Query String Parameters

Name Description
currentPage The current page of results to return. Starts at 1.

JSON POST Parameters

None

Example

https://www.bungie.net/Platform/Group/39966/Members/PendingV2/

{
    "Response": {
        "results": [],
        "totalResults": "0",
        "hasMore": false,
        "query": {
            "itemsPerPage": 10,
            "currentPage": 1
        },
        "useTotalResults": true
    },
    "ErrorCode": 1,
    "ThrottleSeconds": 0,
    "ErrorStatus": "Success",
    "Message": "Ok",
    "MessageData": {}
}
