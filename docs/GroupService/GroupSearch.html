GroupService / GroupSearch

Edit Wiki

Info

Searches for groups matching input string. Returns matching group information.

  • URI: /Group/Search/
  • Method: POST
  • Accessibility: Public
  • Service: GroupService

Parameters

Path Parameters

None

Query String Parameters

Name Description
populatefriends true or false.

JSON POST Parameters

Name Description
contents[searchValue]  
contents[searchType]  
creationDate  
currentPage The current page. Starts at 1.
groupMemberCountFilter  
itemsPerPage Number of results per page. Default is 10.
localeFilter  
membershipType  
sortBy Default 0.
tagText  

Example

POST https://www.bungie.net/Platform/Group/Search/

{
    "contents": {
        "searchValue": "HeisenbergClan",
        "searchType": 0

    "creationDate": 0,
    "sortBy": 0,
    "itemsPerPage": 10,
